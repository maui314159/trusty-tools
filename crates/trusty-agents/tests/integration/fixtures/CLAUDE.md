# Bake-off Integration Test Project

This is a test project for validating open-mpm installation.

## Project Type
Python — FastAPI REST API with pytest tests

## Stack
- Python 3.11+
- FastAPI with Pydantic v2
- pytest with pytest-asyncio
- uv for dependency management
- SQLite for persistence

## Agent Preferences
Use the python-engineer agent for implementation tasks.
Use the qa-agent for test validation.
