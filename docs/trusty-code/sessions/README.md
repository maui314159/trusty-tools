# Session Summaries

This folder contains session-summary documents that capture the trajectory,
decisions, and empirical findings of individual engineering sessions on
trusty-code.

## Purpose

Session documents provide:

- **Narrative context** — the arc from problem statement to shipped features
- **Decision rationale** — why each architectural choice was made in sequence
- **Empirical evidence** — benchmark results and findings that shaped each
  decision
- **Prioritized backlogs** — what shipped, what is blocked, and why

These are distinct from [`regression-testing/`](../regression-testing/) (which
tracks metrics over time) and [`research/`](../research/) (which capture
isolated investigation artifacts).

## Naming Convention

```
SESSION-<YYYY-MM-DD>-<topic>.md
```

Example: `SESSION-2026-07-01-tcode-phase0-scaffold.md`
