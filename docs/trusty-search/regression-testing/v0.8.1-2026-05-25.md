## Anomalies & Context

**Intent Migration (#119)**: This release had the groundwork committed for intent detection (status fields added to the schema), but the runtime hydration was deferred. All queries show `intent=Unknown`, which explains the slight boost in recall (no filtering) but limits quality ranking.

**Two queries below top-1**:
- `type Result`: Generic type alias, appears in many files (namespace pollution)
- `derive(Error)`: Macro invocation, requires understanding source origin (trusty-common)

Both would benefit from intent context once #119 is complete.

## Caveat: BM25 Circular Bias (#123)

‚ö†ÔłŹ **All results in this snapshot are contaminated by BM25 circular bias**:

The trusty-tools repository contains the 14-query suite in `crates/trusty-search/tests/baseline_trusty_tools.rs` as literal strings. When trusty-tools is indexed, BM25 matches the queries directly, artificially inflating Hit@1 and Hit@5.

**Mitigation**: Once #123 lands (queries isolated from the corpus), all baselines will be re-measured with clean data. Until then, treat Hit@1 and Hit@5 as **rough proxies**, not absolute ground truth. Trending (v0.8.1 vs. v0.8.3 vs. future releases) is still valid because the bias is consistent across versions.
